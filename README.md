# [ NEST ] mongo template

1. Clone repository
2. Install dependencies

```bash
yarn
# or
npm Install
```

3. Rename `.template.env` to `.env` and set environment variables to use
4. Run `docker compose up` to run backend and mongo database to start develop

## Stack

- Nest
- Mongo DB
- Passport for authentication
